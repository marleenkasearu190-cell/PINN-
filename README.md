# PINN-

基于 `ERA5`、`MODIS LST` 和 `PINN (Physics-Informed Neural Network)` 的湖泊温度建模与数据处理项目。

这个仓库目前保留了三个阶段的代码：

- [`第二版`](./第二版)：面向 `PINN + PPO + 物理约束 + 纯预测控制` 的实验版目录
- [`第一版`](./第一版)：默认使用的当前版本，也是后续继续开发和维护的主目录
- [`第零版`](./第零版)：早期整合脚本，仅作为历史留档保留

## 默认使用说明

当前仓库默认使用 [`第一版`](./第一版)。

- 如果你想查看 `PINN + PPO` 联合控制、checkpoint 推理和纯预测相关实验，请优先阅读 [`第二版`](./第二版)
- 新功能开发默认放在 [`第一版`](./第一版)
- 日常运行、阅读和后续整理默认以 [`第一版`](./第一版) 为准
- [`第零版`](./第零版) 仅用于回看早期实现，不作为默认入口

## 项目内容

仓库围绕湖泊温度研究的完整流程展开，主要包括：

- ERA5 数据下载、转换与提取
- MODIS 地表温度数据下载
- 湖泊温度相关数据的清洗与整理
- 原始数据和结果数据的可视化
- PINN 模型训练与预测
- 模型结果与观测数据的对比验证

## 版本说明

### 第一版

[`第一版`](./第一版) 是当前默认使用的版本，也是后续更推荐继续整理的版本，分成两个模块：

- `下载爬取/`
- `数据处理/`

其中代表性脚本包括：

- `下载爬取/ERA5下载转化一体.py`
- `下载爬取/ERA5-NC-数据爬取.py`
- `下载爬取/lst下载.py`
- `下载爬取/Mendota真实相关数据下载.py`
- `数据处理/可视化.py`
- `数据处理/预测模型.py`
- `数据处理/验证对比.py`

这一版更适合作为后续重构、补充文档和继续开发的基础。

### 第零版

[`第零版`](./第零版) 保存了较早的一套集中式脚本，主要文件包括：

- `pinn.py`
- `数据爬取.py`
- `提取.py`
- `物理参数.py`
- `原始下载数据可视化.py`
- `下载提取一体.py`
- `运行Mendota流程.ps1`
- `运行Mendota流程.bat`

这个目录更适合用来回看早期方案、原始流程和脚本组织方式，不作为默认使用版本。

## 建议阅读顺序

如果你是第一次进入这个仓库，建议按下面顺序理解：

1. 先看 [`第一版`](./第一版) 的目录结构
2. 再看 `下载爬取` 和 `数据处理` 两部分如何衔接
3. 只有在需要参考旧实现时，再回看 [`第零版`](./第零版)
