# PINN-

基于 `ERA5`、`MODIS LST` 和 `PINN (Physics-Informed Neural Network)` 的湖泊温度建模与数据处理项目。

这个仓库目前保留了两个阶段的代码：

- [`第零版`](./第零版)：早期整合脚本，适合查看最初的整体流程
- [`第一版`](./第一版)：按功能拆分后的版本，结构更清晰，适合后续继续整理和开发

## 项目内容

仓库围绕湖泊温度研究的完整流程展开，主要包括：

- ERA5 数据下载、转换与提取
- MODIS 地表温度数据下载
- 湖泊温度相关数据的清洗与整理
- 原始数据和结果数据的可视化
- PINN 模型训练与预测
- 模型结果与观测数据的对比验证

## 版本说明

### 第零版

[`第零版`](./第零版) 保存了较早的一套集中式脚本，主要文件包括：

- `pinn.py`

- `提取.py`
- `物理参数.py`
- `原始下载数据可视化.py`
- `下载提取一体.py`

这个目录更适合用来回看早期方案、原始流程和脚本组织方式。

### 第一版

[`第一版`](./第一版) 是目前更推荐阅读和继续整理的版本，分成两个模块：

- `下载爬取/`
- `数据处理/`

其中代表性脚本包括：

- `下载爬取/ERA5下载转化一体.py`
- `下载爬取/ERA5-NC-数据爬取.py`
- `下载爬取/lst下载.py`
- `下载爬取/Mendota真实相关数据下载.py`
- `数据处理/可视化.py`
- `数据处理/预测模型.py`
- `数据处理/验证对比.py`

这一版更适合作为后续重构、补充文档和继续开发的基础。

## 建议阅读顺序

如果你是第一次进入这个仓库，建议按下面顺序理解：
