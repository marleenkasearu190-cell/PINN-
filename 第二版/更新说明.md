# 第二版更新说明

本目录新增了第二版核心脚本：

- `PPO策略调控.py`

## 本次更新内容

这一版围绕 `PINN + PPO` 的湖泊水温剖面预测流程进行了整理，重点放在：

- 引入 PPO 策略调控，用于在训练阶段和预测阶段动态调整控制量
- 保留 PINN 的物理约束主干，包括 PDE、边界条件、初始条件与观测项
- 增强对湖泊季节过程的刻画，例如：
  - 冬季逆温分层
  - 夏季分层与温跃层
  - 秋季翻混
- 支持滚动预测、Kalman 滤波和 PPO 在线/离线控制逻辑
- 支持保存与加载 `PINN model checkpoint`，便于续训和纯预测推理

## 适用说明

这一版适合用于：

- 在已有 ERA5、LST 和剖面观测数据下进行训练
- 基于训练后的 checkpoint 执行纯预测
- 后续继续做迁移学习、PPO 调参与物理约束改进

## 与第一版的关系

- 第一版更偏向已有流程与基础数据处理
- 第二版更偏向 `PINN + PPO + 物理约束 + 纯预测控制` 的联合实验框架

## 建议

- 如果要继续开发 PPO、迁移学习和物理约束调度，建议优先在本目录基础上迭代
- 如需保留历史流程，可继续参考仓库中的第一版目录
